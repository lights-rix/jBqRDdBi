# 项目简介

## 核心功能
- 前后端代码及SQL文件
- 万字文档
- 参考导入调试视频
- 讲解视频
- 开发工具安装包

## 适用场景
- 家乡旅游宣传
- 旅游信息展示
- 门票预订
- 旅游攻略查询
- 酒店预订
- 客房预订
- 退订服务
- 举报记录管理
- 论坛分类及交流
- 系统管理
- 个人中心

## 技术特点
- 架构：B/S、MVC、Web框架
- 开发环境：IDEA、JDK1.8、Mysql、Maven、Navcat
- 技术栈：Java、SpringBoot、Vue
